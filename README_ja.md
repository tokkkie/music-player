# Music Player

Wails + React + TypeScript で実装したmp3playerです

世の中のmp3playerの機能を見ると欲しいものがなかったり、不要な機能がついている事も多く、設定も面倒でした。
このソフトウェアはローカルディレクトリの通りに表示を行い、アルバムアートと歌詞だけを表示、再生するだけのシンプルなプレーヤーです。

将来別の機能が欲しくなるかもしれませんが、おそらくこれで十分です。


## 機能

- **アーティストパネル**: アーティスト一覧をA-Z順で表示（"The"を除いてソート）
- **アルバムパネル**: 選択したアーティストのアルバムを年代順で表示
- **トラックリストパネル**: 選択したアルバムの曲リストとアルバムアートを表示
- **歌詞パネル**: ID3タグまたはLRCLIB APIから歌詞を取得して表示
- **プレイヤーパネル**: 再生/一時停止/停止、シークバー、音量調整、リピート機能（1曲/アルバム）
- **設定**: 音楽ディレクトリの選択、フォントサイズ調整（80%〜150%）
- **状態保持**: パネル幅、最後に選択したアーティスト/アルバム、フォントサイズをlocalStorageに保存

## インストール

### ビルド済みバイナリ

[Releases](https://github.com/tokkkie/music-player/releases) からダウンロードしてください。

### ソースからビルド

[Wails](https://wails.io/) のインストールが必要です。

```bash
wails build
```

## 使い方

1. 起動後、アーティストパネルの ⚙️ アイコンから音楽ディレクトリを選択
2. アーティスト → アルバム → 曲をダブルクリックで再生
3. パネル幅はドラッグで調整可能（設定は自動保存）

## 音楽ファイルの配置

音楽ファイルは以下の構造で配置してください:

```text
/path/to/music/
├── アーティスト名/
│   ├── アルバム名/
│   │   ├── 01 曲名.mp3
│   │   ├── 02 曲名.mp3
│   │   └── ...
│   └── ...
└── ...
```
